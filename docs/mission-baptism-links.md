# Obra misional + enlace público de bautismo (24h)

## Publicar enlace 24h
1. Ir a **Obra Misional → Bautismos**.
2. Abrir el servicio bautismal.
3. Como `mission_leader`, usar **Publicar enlace 24h**.
4. El sistema mantiene el `slug` estable NFC (`/bautismo/:slug`) y crea un `code` nuevo para `/b/:slug?c=...`.

## Moderar felicitaciones
1. Ir a **Obra Misional → Moderación**.
2. Revisar cola `pending`.
3. Aprobar o rechazar cada post.
4. Solo posts `approved` aparecen en el muro público.


## Mínimo listo T-14
- Revisa en la pestaña de bautismos el estado **Mínimo listo** por servicio.
- Si está pendiente, completa el programa mínimo, asignaciones críticas y milestone de entrevista programada.
- El job de recordatorios escala en ventanas T-14, T-10, T-7, T-2 y T-1 con dedupe por servicio+regla.
